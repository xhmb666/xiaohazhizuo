# 怀星 · 地铁逃生公告网站 PRD

## 1. 产品概述
面向「地铁逃生」玩家的实时公告发布平台，管理员可跨设备发布/管理公告，普通玩家可对公告进行赞成或反对投票。
- 解决问题：需要一个跨设备同步、仅管理员可发布、玩家可反馈的轻量级公告系统
- 目标用户：地铁逃生玩家群体（浏览者）+ 站长（管理员）

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通访客 | 无需注册 | 浏览公告、投票（赞成/反对） |
| 管理员 | 隐藏密码验证 | 发布、删除公告；查看投票统计 |

### 2.2 功能模块
1. **公告展示页**：品牌标识、公告流、投票交互、页脚联系信息
2. **管理员入口**（隐藏）：左上角「怀星」点击 5 次唤起密码弹窗，验证后进入管理面板
3. **管理面板**：发布公告（标题+正文+可选标签）、删除公告、查看每条投票数

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 公告展示页 | 顶部品牌区 | 「怀星」标识 + 副标题「地铁逃生公告」，点击 5 次唤起密码弹窗 |
| 公告展示页 | 公告卡片流 | 按时间倒序展示公告，含标题、正文、时间戳、赞成/反对按钮及实时计数 |
| 公告展示页 | 页脚 | 小字「企鹅交流：1043816309」 |
| 管理面板 | 发布表单 | 标题、正文、可选紧急标签，提交后实时同步到所有设备 |
| 管理面板 | 公告管理列表 | 列出所有公告，支持删除，显示投票统计 |

## 3. 核心流程

**普通访客流程**：进入网站 → 浏览公告流 → 点击赞成/反对（同一设备同一公告仅可投一次，可切换）→ 投票计数实时更新

**管理员流程**：进入网站 → 连续点击左上角「怀星」5 次 → 弹出密码输入框 → 输入正确密码 → 进入管理面板 → 发布/删除公告

## 4. 用户界面设计

### 4.1 设计风格
**美学方向**：战术广播 / 应急通讯终端美学（Tactical Broadcast Terminal）
- 主色调：深炭黑 `#0a0a0b` 背景 + 琥珀警示橙 `#ff8a3d` 主强调色
- 次要色：血红 `#e23636`（反对）、电光绿 `#3dd68c`（赞成）、冷灰 `#1a1a1d`（卡片）
- 按钮风格：直角带边框、内嵌阴影、hover 时辉光扩散
- 字体：标题用 `Oswald`（压缩感工业字体）+ 正文用 `JetBrains Mono`（终端等宽字体）
- 布局：单列卡片流，顶部固定品牌栏，底部固定页脚
- 装饰：扫描线纹理、CRT 微辉光、品牌区微闪烁、卡片入场错位动画

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|----------|
| 公告展示页 | 品牌栏 | 左上角「怀星」大字 + 副标题，扫描线背景，点击波纹反馈 |
| 公告展示页 | 公告卡片 | 深色卡片、左侧色条标识紧急等级、标题+正文+时间戳+投票区 |
| 公告展示页 | 投票区 | 赞成（绿）/反对（红）双按钮，中间显示净票数，已投票态高亮 |
| 公告展示页 | 页脚 | 居中小字「企鹅交流：1043816309」，淡化处理 |
| 管理面板 | 发布表单 | 模态/抽屉式面板，深色表单，琥珀色提交按钮 |
| 管理面板 | 管理列表 | 表格化展示，每行带删除按钮和投票统计 |

### 4.3 响应式
桌面优先，移动端自适应（卡片宽度 100%、按钮触控优化、品牌字号缩放）

### 4.4 动态效果
- 页面加载：卡片错位淡入（staggered reveal）
- 品牌区：扫描线持续移动、微闪烁
- 投票：点击后数字翻滚动画 + 按钮辉光
- 管理员点击：第 5 次点击时屏幕短暂扫描特效
- 公告新增：从顶部滑入并高亮 2 秒
